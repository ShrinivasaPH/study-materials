🎯 What is PySpark?

PySpark is the Python API for Apache Spark, a powerful open-source distributed computing framework designed for fast and large-scale data processing.

Why PySpark?

  • Speed: Spark processes data up to 100x faster than Hadoop MapReduce
  • Easy to Use: Simple, intuitive API similar to Pandas
  • Distributed Computing: Handle big data across multiple machines
  • Flexible: Works with multiple data sources (HDFS, S3, Cassandra, etc.)
  • Python Integration: Use Python's rich ecosystem for data science

Key Concepts

RDD (Resilient Distributed Dataset): The fundamental data structure of Spark. An immutable, distributed collection of objects that can be processed in parallel.
DataFrame: A distributed collection of data organized into named columns. Similar to a Pandas DataFrame but distributed.
Transformations: Operations that create a new RDD/DataFrame from an existing one (lazy evaluation).
Actions: Operations that return values to the driver program or write data to storage.

Spark Architecture

Spark follows a master-slave architecture:

  • Driver Program: Runs your main code and creates SparkContext
  • Cluster Manager: Allocates resources (Standalone, Yarn, Mesos)
  • Worker Nodes: Run executor tasks in parallel

⚙️ Installation & Setup

Prerequisites

  • Java (JDK 8 or higher)
  • Python (3.6 or higher)
  • Scala (optional, for advanced usage)

Install PySpark

The easiest way to install PySpark is using pip:

pip install pyspark

Create Your First Spark Session

A SparkSession is the entry point to PySpark functionality:

from pyspark.sql import SparkSession # Create a SparkSession spark = SparkSession.builder \ .appName("MyApp") \ .master("local[*]") \ .getOrCreate() # Get SparkContext (for RDD operations) sc = spark.sparkContext
Note: .master("local[*]") uses all available cores on your local machine. For production, use a cluster manager like YARN or Kubernetes.

Testing Installation

# Simple test to verify installation rdd = sc.parallelize([1, 2, 3, 4, 5]) result = rdd.map(lambda x: x * 2).collect() print(result) # Output: [2, 4, 6, 8, 10]

📦 RDDs (Resilient Distributed Datasets)

What are RDDs?

RDDs are the fundamental data structure of Apache Spark. They are immutable distributed collections of objects that can be processed in parallel across a cluster.

Creating RDDs

Method 1: From a Python Collection

data = [1, 2, 3, 4, 5] rdd = sc.parallelize(data)

Method 2: From External Storage

# Read from HDFS or local file rdd = sc.textFile("path/to/file.txt")

Method 3: From Another RDD

rdd2 = rdd.map(lambda x: x * 2)

Basic RDD Operations

# Create RDD rdd = sc.parallelize([1, 2, 3, 4, 5]) # Map: apply function to each element squared = rdd.map(lambda x: x ** 2) # Filter: keep elements that satisfy condition evens = rdd.filter(lambda x: x % 2 == 0) # Reduce: aggregate elements sum_rdd = rdd.reduce(lambda a, b: a + b) # Collect: return all elements to driver result = rdd.collect() print(result)
Example: Find sum of squares of even numbers
result = rdd \ .filter(lambda x: x % 2 == 0) \ .map(lambda x: x ** 2) \ .reduce(lambda a, b: a + b) print(result) # Output: 36

📊 DataFrames

What are DataFrames?

DataFrames are distributed collections of data organized into named columns. They are similar to Pandas DataFrames but distributed across a cluster, allowing for fast processing of large datasets.

Creating DataFrames

Method 1: From Python Data

data = [("Alice", 25), ("Bob", 30), ("Charlie", 35)] columns = ["name", "age"] df = spark.createDataFrame(data, schema=columns) df.show()

Method 2: From Dictionary/List of Rows

from pyspark.sql import Row data = [ Row(name="Alice", age=25), Row(name="Bob", age=30) ] df = spark.createDataFrame(data)

Method 3: From CSV, JSON, Parquet

# Read CSV df = spark.read.csv("data.csv", header=True) # Read JSON df = spark.read.json("data.json") # Read Parquet df = spark.read.parquet("data.parquet")

DataFrame Operations

# Display data df.show() # Get schema df.printSchema() # Select columns df.select("name", "age").show() # Filter rows df.filter(df.age > 25).show() # Get statistics df.describe().show() # Count rows print(df.count())

🔍 SQL Operations

Using SQL with PySpark

PySpark allows you to write SQL queries directly on DataFrames. First, register the DataFrame as a temporary table:

# Register DataFrame as temporary table df.createOrReplaceTempView("users") # Write SQL query result = spark.sql(""" SELECT name, age FROM users WHERE age > 25 ORDER BY age DESC """) result.show()

Common SQL Operations

# SELECT with WHERE spark.sql("SELECT * FROM users WHERE age > 30").show() # GROUP BY and aggregation spark.sql("SELECT age, COUNT(*) as count FROM users GROUP BY age").show() # JOIN spark.sql(""" SELECT u.name, o.order_id FROM users u JOIN orders o ON u.id = o.user_id """).show() # UNION spark.sql("SELECT * FROM table1 UNION SELECT * FROM table2").show()
Tip: You can mix SQL and DataFrame API. Both have similar performance as Spark optimizes them using Catalyst optimizer.

⚡ Transformations

What are Transformations?

Transformations are operations that create a new RDD/DataFrame from an existing one. They are lazy, meaning they don't compute results immediately.

Common DataFrame Transformations

1. Select

# Select specific columns df.select("name", "age").show()

2. Filter (Where)

# Keep rows matching condition df.filter(df.age > 25).show() df.where(df.salary > 50000).show()

3. Map (For RDDs)

rdd = sc.parallelize([1, 2, 3, 4]) mapped = rdd.map(lambda x: x * 2)

4. FlatMap

# Map then flatten results rdd = sc.parallelize(["Hello World", "PySpark"]) words = rdd.flatMap(lambda x: x.split(" ")) # Result: ["Hello", "World", "PySpark"]

5. Union

df3 = df1.union(df2)

6. Join

df_joined = df1.join(df2, df1.id == df2.id)

7. GroupBy

# Group and aggregate df.groupBy("department").count().show() df.groupBy("age").agg({"salary": "avg"}).show()
Key Point: Transformations are lazy! They don't execute until you call an action like collect(), show(), or save().

🎬 Actions

What are Actions?

Actions are operations that return values to the driver program or write data to storage. They trigger the execution of transformations (lazy evaluation).

Common Actions

1. Collect

Returns all elements to the driver program (use carefully with large datasets):

result = df.collect() print(result)

2. Show

Display first n rows:

df.show(5) # Show first 5 rows df.show(False) # Don't truncate columns

3. Count

Return number of rows:

count = df.count() print(f"Total rows: {count}")

4. First / Head

Get first row(s):

first_row = df.first() head_rows = df.head(3)

5. Take

Get first n elements (from RDD):

rdd = sc.parallelize([1, 2, 3, 4, 5]) first_three = rdd.take(3) # [1, 2, 3]

6. Reduce

Aggregate elements:

rdd = sc.parallelize([1, 2, 3, 4, 5]) sum_val = rdd.reduce(lambda a, b: a + b)

7. Write / Save

Write DataFrame to storage:

# Save as CSV df.write.csv("output.csv", header=True) # Save as Parquet df.write.parquet("output.parquet") # Save as JSON df.write.json("output.json")
Example: Action triggering transformations
# These are lazy - nothing happens yet df2 = df.filter(df.age > 25) df3 = df2.select("name") # This action triggers execution of all transformations df3.show()

🌍 Real-World Example

Analyzing Sales Data

Let's build a complete example that demonstrates PySpark concepts:

from pyspark.sql import SparkSession from pyspark.sql.functions import col, avg, count, max # Initialize Spark spark = SparkSession.builder.appName("SalesAnalysis").master("local[*]").getOrCreate() # Create sample data data = [ ("Alice", "Electronics", 500), ("Bob", "Electronics", 1200), ("Charlie", "Clothing", 300), ("David", "Clothing", 450), ("Eve", "Electronics", 800), ] columns = ["customer", "category", "amount"] df = spark.createDataFrame(data, schema=columns) # 1. Display data df.show() # 2. Filter high-value transactions high_value = df.filter(df.amount > 600) high_value.show() # 3. Group by category and aggregate summary = df.groupBy("category") \ .agg( count("*").alias("count"), avg("amount").alias("avg_amount"), max("amount").alias("max_amount") ) summary.show() # 4. Using SQL df.createOrReplaceTempView("sales") top_customers = spark.sql(""" SELECT customer, category, amount FROM sales WHERE amount > 500 ORDER BY amount DESC """) top_customers.show() # 5. Save results summary.write.csv("sales_summary.csv", header=True)
